# ytdata

**Y**ou**t**ube **data** parser

## Features

This parser gets:
- Video info
  - [ ] Title
  - [ ] Description
  - [ ] Publish date
  - [ ] Channel name
- Video data
  - [ ] Subtitles
  - [ ] Comments
    - [ ] Comments of comments
  - Livechat
    - [ ] Realtime
    - [ ] Replay
- [ ] Search results
- [ ] All videos from channel/user
- [ ] All videos of playlist (a seq of videos)
